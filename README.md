# La Biga

Landing page para **La Biga**, panadería artesanal de Buenos Aires especializada en ciabatta con fermentación lenta de 24h y masa madre.

## URL

**Producción:** https://gigisanta.github.io/labiga/

## Tech Stack

- HTML5 vanilla
- Tailwind CSS (CDN)
- Google Fonts (Playfair Display + Inter)
- Sin dependencias de build — deploy directo por GitHub Pages

## Estructura

```
index.html    # Single-page landing
README.md
```

## Deploy

GitHub Pages desde la rama `master`, path `/`.
Cualquier push a `master` se refleja en producción en segundos.
